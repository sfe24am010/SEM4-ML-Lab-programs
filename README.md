# 🤖 Machine Learning Lab Programs – SEM 4

A curated collection of SEM 4 Machine Learning lab experiments covering fundamental and advanced ML algorithms with Python implementation, dataset preprocessing, model training, evaluation metrics, and visualization. Designed for academic excellence and practical ML mastery.

## 📌 Program List

1. House Price Prediction – Linear Regression  
2. Student Performance Prediction – Regression Model  
3. Diabetes Detection – Logistic Regression  
4. Handwritten Digit Recognition – KNN Algorithm  
5. Flower Classification – Decision Tree  
6. Customer Churn Prediction – Random Forest  
7. Email Spam Classification – Support Vector Machine  
8. Customer Segmentation – K-Means Clustering  
9. Market Basket Analysis – Apriori Algorithm  
10. Movie Recommendation System – Collaborative Filtering
